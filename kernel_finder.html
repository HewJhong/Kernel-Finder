Interactive tool to find all kernels in directed graphs

Graph Input
Format: First line: "n m" (vertices edges). Then for each vertex: "j1 j2 ... jd : i" where j1,j2,...,jd are in-neighbors of vertex i.
Example:
4 4
2 : 1
1 3 : 2
: 3
2 : 4
This represents a 4-vertex graph where vertex 1 has in-neighbor 2, vertex 2 has in-neighbors 1 and 3, vertex 3 has no in-neighbors, and vertex 4 has in-neighbor 2.
